# 凭证与仓库操作说明

## 1. 当前环境限制

当前这个运行环境可以读取公开仓库页面，也可以在本地生成代码与压缩包，
但**没有可直接写入 GitHub 仓库的能力**，所以不能在这里替你执行 `git push`。

## 2. 关于 GitHub Token

你在聊天里贴出的 GitHub Personal Access Token 已经属于**暴露凭证**。
建议你现在就：

1. 在 GitHub 上立刻 **revoke** 当前 token。
2. 重新生成一个新的 token。
3. 以后不要把 token 直接贴到聊天里，优先通过本机环境变量注入。

## 3. 推荐的本地推送方式

在你自己的电脑上进入 `suitme-pro` 仓库后执行：

```bash
git checkout -b feat/auth-common-foundation
# 把本次更新包里的文件覆盖到仓库工作区
git add .
git commit -m "feat: implement auth/common foundation for fastapi migration"
git push origin feat/auth-common-foundation
```
