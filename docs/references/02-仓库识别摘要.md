
# 仓库识别摘要

这份摘要是为了给本地继续开发时快速回忆上下文。

## 已确认的仓库事实

- 项目基于若依后端改造。
- 当前公开仓库同时包含 `ruoyi-ui`、`ruoyi-admin`、`ruoyi-framework`、`ruoyi-system`、`ruoyi-quartz`、`ruoyi-generator`、`sql`、`doc` 等目录。
- 项目里包含 SUITME 自定义业务：顾客、搭配、穿搭生图、标签、服装分类树、OSS。
- 当前鉴权头是 `Authorization`，JWT 返回字段名是 `token`。
- OpenAPI 地址是 `/swagger-ui.html` 和 `/v3/api-docs`。
- 外部 AI 服务基地址默认是 `http://127.0.0.1:9001`。
- 注册成功后会初始化默认服装分类树。

## 本地继续接手时优先核对

1. 真实数据库表结构。
2. 全部 Controller 的权限字符串。
3. 文件上传下载的真实行为。
4. 任务状态流转细节。
