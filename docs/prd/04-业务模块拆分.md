
# 业务模块拆分

## 1. 顾客模块

- `/customer/add`
- `/customer/get`
- `/customer/getDigitalImg`
- `/customer/page`
- `/customer/delete`
- `/customer/update`

关键点：新增顾客后会立即调用外部 AI 服务生成数字形象。

## 2. 搭配模块

- `/matching/addOrUpdate`
- `/matching/list`
- `/matching/delete`

关键点：搭配主表和搭配 SKU 明细要一起维护。

## 3. 搭配标签模块

- `/matching/tag/addOrUpdate`
- `/matching/tag/list`
- `/matching/tag/delete`

## 4. 穿搭生图模块

- `/outfit/generateOutfitImg`
- `/outfit/getOutfitImg`
- `/outfit/getTaskStatus/{taskId}`
- `/outfit/taskPage`

关键点：`ai_join`、`ai_outfit`、`ai_task` 三类表的状态流转要和 Java 保持一致。
