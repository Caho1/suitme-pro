
# 外部集成

## AI 生图服务

当前骨架已提供 `app/services/ai_client.py`，预留三个方法：

- `generate_digital_image()`
- `generate_outfit_image()`
- `get_task_status()`

## OSS

当前骨架预留了 OSS 服务包装层，后续只需要把：

- 文件 key 生成规则
- 上传后数据库落表
- 下载时 content-type / content-disposition

对齐 Java 即可。

## 本地上传目录

`/common/upload` 和 `/common/uploads` 仍按本地磁盘处理，不强行改成 OSS。
